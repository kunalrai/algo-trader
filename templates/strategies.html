Strategy Structure 

from strategies.base_strategy import BaseStrategy

class MyCustomStrategy(BaseStrategy):
    """Your strategy description here"""

    def __init__(self, min_strength=0.6):
        super().__init__()
        self.min_strength = min_strength

    def get_required_timeframes(self):
        """Return list of timeframes needed"""
        return ['5m', '1h', '4h']

    def get_required_indicators(self):
        """Return list of indicators needed"""
        return ['ema_9', 'ema_21', 'rsi', 'macd']

    def analyze(self, data, current_price):
        """Analyze market data and return signal"""
        # Your trading logic here
        return {
            'action': 'long',  # 'long', 'short', or 'flat'
            'strength': 0.8,   # 0.0 to 1.0
            'confidence': 0.75,
            'reasons': ['EMA crossover detected'],
            'indicators': {},
            'metadata': {}
        }

Strategy Guidelines

Changing Strategies

When you change strategies, the bot will restart its analysis cycle and use the new strategy for all future signal generation.

Existing Positions

Open positions are not affected by strategy changes. They will continue to follow their original exit rules.

Testing Strategies

In DRY RUN mode, you can safely test different strategies without risking real funds. Monitor performance before switching to live trading.
